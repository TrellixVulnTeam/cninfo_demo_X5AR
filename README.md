# finance_sys_demo
a demo of a web_based visualization application of finance data

运行：
python app.py

## main 

**是主要逻辑部分**

- conv_node_edge.py 转换节点和边
- conv_table.py 转换查询结果为表格
- query.py 处理查询
- template_search.py 一些简单的模板查询

## static 

**是界面相关部分**

- graph.js ecahrts初始化、点边处理和其他
